# Ejercicios del Curso

Ejercicios prácticos organizados por fase y unidad. Cada unidad tiene un archivo de **ejercicios** (para resolver) y un archivo de **soluciones** (para verificar).

---

## Cómo usar los ejercicios

1. Abre el archivo `UXX_ejercicios.py` en tu editor (VS Code)
2. Lee el enunciado y escribe tu solución donde dice `# Escribe tu código aquí`
3. Corre el archivo y verifica que tu output coincida con el `# Output esperado`
4. Compara tu solución con `UXX_soluciones.py`
